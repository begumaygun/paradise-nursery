Project Overview
Paradise Nursery is a React-based shopping cart application designed for an online plant shop. It allows users to browse plants, view details, add items to a cart, update quantities, and view the total price dynamically.

Features
Plant listing with images, prices, and descriptions
Add to cart functionality
Increase or decrease quantities
Remove items from cart
Dynamic total cost calculation
Responsive and user-friendly UI
Technologies Used
React
JavaScript (ES6)
HTML5
CSS3
